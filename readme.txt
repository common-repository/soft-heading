Stylish Sub Heading for WordPress - To show custom styled Heading Bar 

== Description ==
WordPress Heading Plugin lets you to Display Superb Coloured Style Heading Bar with the Help of Shortcodes.

> Usage:
You can Add New Stylish Heading with the Help of Button in Visual Editor.

WordPress Heading Plugin lets you to Display Superb Coloured Style Heading Bar with the Help of Shortcodes. It shows the Different Between the Normal Paragraphs and Heading. It wont Affect your SEO.

- You Can Choose your Own Color
- Full Featured Control Panel
- Responsive & Very Flexible
- Works with All themes

If you Have Any Questions Related in this Plugin Please Use the WordPress forum and I will take care of any issue.

== Installation ==
Upload the soft_heading folder to the /wp-content/plugins/directory

Activate the plugin through the \'Plugins\' menu in WordPress

== Usage ==
> You can Add New Stylish Heading with the Help of Button in Visual Editor
